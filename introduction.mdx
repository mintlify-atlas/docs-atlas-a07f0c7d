# Welcome to MC World Compressor Backend

MC World Compressor is a powerful Laravel 12 API backend that efficiently compresses Minecraft worlds using the [Thanos](https://github.com/aternosorg/thanos) library. This tool is designed to reduce file sizes for backups, world transfers, and storage optimization.

## How It Works

The MC World Compressor backend processes Minecraft worlds in three main steps:

1. **Upload**: Submit your Minecraft world archive (ZIP or TAR.GZ) via the API
2. **Process**: The backend extracts, analyzes, and removes unused chunks using Thanos
3. **Download**: Receive your compressed world archive ready for use

<Note>
The compression process removes chunks with an InhabitedTime of 0, which are chunks that have never been visited by players. This significantly reduces world file size without affecting gameplay.
</Note>
